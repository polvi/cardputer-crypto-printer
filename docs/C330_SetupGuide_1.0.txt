[1 – Embosser Configuration]

The parameters in the Embosser Configuration are:
Machine Model - C330 mode Factory defined
Machine Serial Number – SN Factory defined
Type of embossing
o STANDARD – to set standard emboss fonts
o REVERSED – to set revered emboss fonts
Coodinate Value sets the measure unit; the options are: tenth of millimeters hundreds of inches,
thousands of inches or steps.
mm/10 is sets as default.
Feeder type - the options are Step or DC Motor.
Unload type - the options are FIFO or Eject.

[2 – Mechanical Parameters]

The parameters in the Mechanical Parameters are:
Drum HOME: adjust the home position of the drum.
X-Start Emboss: sets the emboss X- axis offset in order to align the format
coordinates with the embossed characters positions.
Y-Start Emboss: sets the emboss Y- axis offset in order to align the format
coordinates with the embossed characters positions.
X-Home: sets the X-axis waiting plate position.
X-End: sets the position where is checked if the X-axis embossing movement has been correct.
Y-Home: sets the Y-axis waiting plate position.
X-Load: sets the X-axis position to grip the metal plate; the machine will first move the Y-axis to load the
metal plate and then the X-axis to close the clamp.
Y-Load: sets the Y-axis position to grip the metal plate; it’s very important to have the metal plate fully
inside the clamp.

C330 Setup Guide

Rev. 1.0

Page 12/22

X-Unload: sets the X-axis position to release the metal plate.
Y-Unload: sets the Y-axis position to release the metal plate.
X Drum Entry: sets the X-axis position where the metal plate will go inside the drum area to ensure an
easier metal plate enter.
Y Drum Entry: sets the Y-axis position where the metal plate will go inside the drum area to ensure an
easier metal plate enter.
X Card size (mm): sets card size value on X Axis
Y Card size (mm): sets card size value on Y Axis
Unload Time out (ms): sets the maximum time to eject a metal tag before generating an error.
Clamp – Y safe zone: set Y steps not used in the clamp to avoid punch the clamp base.
Feeder Offset +CARD IN: sets how much the Loader has to move out in order to load the plate in the
Clamp.
Feeder Offset -CARD IN: set how much the Loader has to come back after the Clamp has taken the plate.

[3 – X Axis]

The parameters in the X axis are:
X Motor RESTORE Speed (Hz): sets the restore movement of the X-axis motor; increasing the value
increases the restore speed.
X Motor Accel: sets the number of steps needed by the X-axis motor for its acceleration ramp.
X Motor Max Speed (Hz): sets the top speed limit of the X-axis motor; increasing the value increases the
limit.

The parameters in the Y axis are:
Y Motor RESTORE Speed (Hz): sets the restore movement of the Y-axis motor; increasing the value
increases the restore speed.
Y Motor Accel: sets the number of steps needed by the Y-axis motor for its acceleration ramp.
Y Motor Max Speed (Hz): sets the top speed limit of the Y-axis motor; increasing the value increases the

[5 – Drum Axis]

The parameters in the Drum axis are:
Drum Motor RESTORE Speed (Hz): sets the restore movement of the Drum -axis motor; increasing the
value increases the restore speed.
Drum Motor Accel: sets the number of steps needed by the Drum -axis motor for its acceleration ramp.
Drum Motor Max Speed (Hz): sets the top speed limit of the Drum -axis motor; increasing the value
increases the limit.

[6– Feeder Axis]

The parameters in the feeder axis are:
Feeder Motor RESTORE Speed (Hz): sets the restore movement of the feeder-axis motor; increasing the
value increases the restore speed.
Feeder Motor Accel: sets the number of steps needed by the feeder-axis motor for its acceleration ramp.
Feeder Motor Max Speed (Hz): sets the top speed limit of the feeder-axis motor; increasing the value
increases the limit.

[7– Punch Axis]

The parameters in the Punch axis are:
Delay Before Punch: delay time before a completing embossed metal tag,
Delay After Punch: delay time after a completing embossed metal tag.

[8– Protocol]

The parameters in the Protocol are:
Protocol: the options are: XONXOFF, Chip, L2, US Open, SEP, ##, Euro1.
Buffer Mode:
• Single if check marked, avoiding the loading of another plate while the working cycle of the already
loaded one has not been completed yet.
•

Multiple if check marked, allows the loading of another plate while the
working cycle of the already loaded one has not been completed yet.

On-Line Format: sets the format to use on-line with a PC between the ones stored into the machine.
The ASCII character of each following data opener, data closer, field separator or other parameter is shown
aside the related box.
The parameters in the Protocol Characters area are related to the XOn/XOff protocol and are disabled:
•
•
•

Value of STX: sets the ASCII value of the protocol data opener character.
Value of ETX: sets the ASCII value of the protocol data closer character.
Value of fields separator:sets the ASCII value of the protocol fields separator character.

The parameters in the US Open Protocol Chars area are disabled:
•

Value of US Open: sets the ASCII value of the code character in the US Open protocol.

•

Value of OCR Start: sets the ASCII value of the OCR data opener character in the US Open

Length Error message: set a value to defined error message length.

Ignore Overflow Characters: if check marked, cut an overflow characters field instead of sending an error.
Illegal Char = Space: if check marked, convert an illegal character in a space instead of sending an error.

[9 – Serial Port]

The parameters in the Serial Port are:
Baudrate: sets the protocol communication speed.
Data Bits + Parity: sets the protocol communication data bits and parity

[10 – Module Test]

The parameters in the Module Test are:
Disable Card Presence: if flagged, the card presence sensor is disable during card production,
Disable X Module: if flagged, the x axis Carriage is disabled.
Disable Y Module: if flagged, the Y axis Carriage is disabled.
Disable Drum Module: if flagged, the Drum is disabled.
Disable Punch Cycle: if flagged, the Emboss Punch cycle is disabled.
Disable Feeder Cycle: if flagged, the Feeder cycle is disabled.

Load /Save machine Setup

Load setup: to load machine configuration from xml file stored in the host PC.
Save setup: to save machine configuration to xml file in the host PC.
